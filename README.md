# aws-s3-bucket-file-upload
Node script to upload file on S3 Bucket

#### list_bucket.js
Script to get list of already created S3 buckets.

#### upload_file.js
Script to upload file on S3 bucket.

#### delete_bucket.js
Script to delete S3 bucket.

### Setup
git clone "---url---"\
npm init\
npm install\
node filename.js "--- to run----"
